_myIncludeAll 用于保存自定义的代码头文件， 在main中引入这一个文件即可


_myMenuBar  用于制作主页菜单
    _myMenuBarAction 用于制作菜单功能函数


_myTabWidget 用于制作主页主界面
    _myPopWidow 用于制作双击主界面时弹出的新窗口
        _myPopTabWindowIndex 用于制作弹出新窗口时的主页
        _myPopTabWindow......其他页面
    _myTabWindowIndex 用于主页index页
    _myTabWindowFofa 用于主页fofa页
    _myTabWindow......用于其他页面


_myTabWidgetContextMenu 用于制作右键菜单， 用于主页


_myPopWindowContextMenu 用于制作弹出页页面的菜单
